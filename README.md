**Tandemloop – First Screening Test Solutions**
  This repository contains the solutions for the First Screening Test as part of the Full Stack Developer Hiring Process at Tandemloop.
The test consists of 4 programming problems, solved using Python as the chosen language.
  **Test Instructions (As Given)**
    Solve all problems using any programming language.
    Upload each solution in a separate file.
    File Names: Problem-1, Problem-2, Problem-3, Problem-4
    Add your GitHub repository link to the Google Form once completed.
Complete the test within 2 days of receiving the mail.

Problem Descriptions and Sample Test Cases

**Problem-1: Calculator Using Class**

Create a calculator using a class that performs the following operations:
Addition
Subtraction
Multiplication
Division

Inputs:
a → double
b → double
operation → string (addition, subtraction, multiplication, division)

Sample Test Case:
Input:
a = 10
b = 5
operation = addition

Output:
15

**Problem-2: Generate Odd Number Series (n Terms)**

Given a single integer a, generate the first a odd numbers.

Sample Test Cases:
Input: 1
Output: 1

Input: 3
Output: 1, 3, 5

Input: 5
Output: 1, 3, 5, 7, 9

**Problem-3: Conditional Odd Number Series**

Given a single integer a:
If a is odd, print odd numbers up to a terms.
If a is even, print odd numbers up to (a - 1) terms.

Sample Test Cases:
Input: 1
Output: 1

Input: 2
Output: 1

Input: 3
Output: 1, 3, 5

Input: 4
Output: 1, 3, 5

Input: 6
Output: 1, 3, 5, 7, 9

**Problem-4: Count Multiples of 1–9**

Given a list of numbers, count how many numbers are multiples of 1, 2, 3, 4, 5, 6, 7, 8, 9.

Sample Input:
[1, 2, 8, 9, 12, 46, 76, 82, 15, 20, 30]

Sample Output:
{1: 11
2: 8
3: 4
4: 4
5: 3
6: 2
7: 0
8: 1
9: 1}
